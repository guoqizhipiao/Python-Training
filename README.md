# 学生人脸识别项目  
**第9组 · 项目仓库**

## ✨ 功能

### 数据库管理
- 支持创建并维护学生照片与个人信息的本地数据库。
- 提供两种数据录入方式：
  - 本地 GUI 管理界面；
  - 基于网页的上传接口。

### 人脸识别系统
- 利用本地训练的人脸识别模型，对输入图像进行身份匹配。
- 识别成功后，自动展示对应学生的详细信息。
- 集成网络爬虫模块，可自动抓取公开人脸图像用于模型训练（遵守合规前提下）。

---

## 📚 相关知识

本项目综合运用以下技术栈：
- **GUI 开发**：`tkinter`
- **计算机视觉**：`OpenCV`
- **数据库操作**：`SQLite3`
- **网络通信**：`socket` 编程
- **网络爬虫**：`BeautifulSoup` + `requests`
- **并发处理**：`threading` 多线程
- **前端基础**：HTML（用于网页上传界面）

---

## ⚙️ 核心架构

- 程序入口
- ├── GUI 模块
- │   ├── 人脸识别界面
- │   └── 数据库管理界面
- └── 核心功能模块
- ├── 数据库操作（SQLite）
- ├── 服务器通信（Socket）
- ├── 人脸识别（OpenCV + 本地模型）
- └── 网络爬虫（自动采集训练数据）
